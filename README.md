## Sobre o Projeto

Na intenção de exercitar e relembrar um pouco mais de algoritmos, desenvolvi um jogo clássico chamado Pong, junto ao curso de Lógica de programação da Alura, usando JavaScript e uma biblioteca do P5.JS.
